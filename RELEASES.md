# Releases

## Latest Release (March 5, 2026)

Current stable version with all features:
- Full file management (read, write, delete, move)
- Complete git integration (10 operations)
- Multi-provider LLM support (Gemini, OpenAI, Claude, Groq)
- Undo functionality
- Secure API key storage

---

## Download

- **PyPI**: `pip install forgecodecli`
- **Windows**: `forgecodecli.exe` (~20.8 MB) - no installation needed

---

## Installation

```bash
pip install forgecodecli
```

**Windows Users**:
Download `forgecodecli.exe` from [GitHub Releases](https://github.com/TiwariSudhanshu/forgecodecli/releases) and run directly - no Python needed.

**From Source**:
```bash
git clone https://github.com/TiwariSudhanshu/forgecodecli.git
cd forgecodecli
pip install -e .
```
